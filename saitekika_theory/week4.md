Homework 8 — Lagrange multipliers & the KKT conditions
minimize f(x) = x₁² + x₂² + x₃²
1 等式制約 x₁ + x₂ + x₃ = 3 のもとで、ラグ
ランジュの未定乗数法により最小解・乗
数 λ・最小値を求めよ。
2 不等式制約 x₁ ≥ 2 を加える。(1) の解はこ
れを満たすか確認し、満たさない場合に
何が起きるかを述べよ。
3 KKT 条件で制約の有効/無効を場合分けし
、最小解と乗数 (λ, μ) を求めよ。μ ≥ 0
と相補性 μ·(2−x₁) = 0 の成立を確認せよ
。
4 〔発展〕乗数 μ の「影の価格」としての
意味を述べよ。制約を x₁ ≥ b に一般化し
たとき、f* が b にどう応答するか論ぜよ
。
ヒント： 不等式は g(x)=2−x₁ ≤ 0 と書ける。停留性 ∇f+λ∇h+μ∇g=0、μ≥0、相
補性 μ·g=0 を連立。まず有効(x₁=2)を仮定して解き、μ≥0 を確認する。
